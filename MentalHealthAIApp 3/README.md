A Streamlit-based self-help and psychoeducation app integrating AI and validated self-assessment tools.

## Features
- AI-based chat for emotional support
- PHQ-9, GAD-7, and WHO-5 screening tools
- Psychoeducational modules (CBT, mindfulness, etc.)
- Mood and progress tracking
- Ethical design with privacy and safety disclaimers

## Setup Instructions
1. Install dependencies:
   ```bash
   pip install streamlit openai
   ```
2. Add your OpenAI API key in `config.toml`.
3. Run the app:
   ```bash
   streamlit run app.py
   ```

## Deployment
You can deploy this on **Streamlit Cloud** or import it into **Lovable** for no-code customization.
